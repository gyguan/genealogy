# MVP Frontend

这是 MVP 1 最小可演示前端，采用纯静态 HTML/CSS/JavaScript 实现，不引入构建链。

## 使用方式

启动后端后，可用任意静态服务器打开本目录：

```bash
cd frontend/mvp
python3 -m http.server 5173
```

浏览器访问：

```text
http://localhost:5173
```

如果后端不是同源地址，需要反向代理 `/api/v1` 到后端服务，或在本地浏览器允许跨域。

## 覆盖页面

- 登录/注册
- 宗族创建与列表
- 支派维护
- 字辈维护
- 人物录入与列表
- 关系维护
- 来源绑定与附件上传
- 审核提交与处理
- 世系查看
- CSV 导入导出
- 操作日志查看
