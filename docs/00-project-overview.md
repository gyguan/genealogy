# 00. 项目概览

## 产品定位

中国式族谱系统不是普通家谱树工具，而是面向宗亲会、修谱委员会、支派负责人、采集员、普通族人的数字化修谱平台。

核心目标是把传统族谱中的宗族组织、支派房支、人物世系、谱书来源、审核定稿、电子查看和简版导出沉淀为可持续维护的数据资产。

## MVP 1 主闭环

```text
创建宗族 → 建立支派 → 维护字辈 → 录入人物 → 建立关系 → 绑定来源 → 提交审核 → 查看世系 → 导出简版族谱
```

## 核心用户

| 角色 | 职责 |
|---|---|
| 宗族管理员 | 创建宗族空间、管理成员、维护宗族主数据 |
| 主编/总纂 | 审核族谱资料、终审定稿、导出谱书 |
| 支派负责人 | 管理本支派资料、处理初审 |
| 采集员 | 录入人物、关系和资料来源 |
| 普通族人 | 查看授权范围内族谱，提交本人家庭资料 |

## 核心对象

- Clan：宗族
- Branch：支派/房支
- Generation：字辈/派语
- Person：人物
- Relationship：人物关系
- Source：资料来源
- Revision：变更记录
- ReviewTask：审核任务

## 建设原则

1. 人物和关系分离，关系作为独立对象。
2. 人物与关系都可以绑定来源证据。
3. 正式数据修改必须走审核流程。
4. 支派权限和在世人员隐私要前置设计。
5. MVP 1 采用模块化单体，后续按业务成熟度拆分。
