# 06. Roadmap

## MVP 1：核心建谱闭环

目标：让一个宗族可以完成基础建谱、人物录入、关系维护、来源绑定、审核流转、世系查看和简版导出。

核心能力：

- 宗族管理
- 支派管理
- 字辈管理
- 人物档案
- 关系管理
- 来源证据
- 审核中心
- 世系图
- Excel 导入导出
- 权限隐私

## MVP 2：协作修谱与数字化增强

目标：提升多人修谱效率，支撑更大规模族谱建设。

计划能力：

- 族人邀请和在线填报
- 批量审核
- 更完善的变更历史
- 老谱图片管理增强
- 基础 OCR 接入
- 重复人物合并
- 支派任务分发
- 电子谱阅读优化

## MVP 3：智能寻祖与谱书出版

目标：形成完整族谱服务闭环。

计划能力：

- AI 人名识别
- AI 关系候选抽取
- 寻祖路径推荐
- 迁徙图
- 五服关系计算
- 谱书目录编排
- PDF 版式输出
- 宗亲社区
- 活动与公告

## 长期演进方向

```text
族谱数据底座 → 协作修谱平台 → 老谱数字化中心 → 谱书出版平台 → 宗亲知识服务平台
```
