## 完成内容

- 页面增加标准标题、说明和当前范围状态。
- 页面拆分为“新建导入 / 执行任务 / 导入记录”三个稳定视图。
- 导入类型改用 `Segmented`，避免嵌套 Tabs。
- Steps 按目标、文件、预检和批次状态真实推进。
- 切换导入类型或目标支派时明确提示旧预检结果失效。
- 上传改用 `Upload.Dragger`，补充格式、20 MB 大小、空文件和模板说明。
- 下载模板、预检和创建批次使用独立 loading 状态。
- 三类导入均在存在错误行时阻断批次创建。
- 人物疑似重复仅在预检后展示确认框；未确认时禁止创建批次。
- 人物预检支持分类筛选、分页和错误明细 CSV 下载。
- 异步任务展示总数、已处理、成功、失败和最近更新时间。
- 完成但存在失败行的任务展示为“部分成功”。
- 行内操作改为文字按钮；取消任务增加二次确认。
- 技术执行字段移入 720 px 任务详情 Drawer。
- 新增导入工作台状态规则测试并纳入 Frontend CI。
