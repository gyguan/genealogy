# Issue #106 来源资料导入执行看板

## 目标

实现来源资料元数据 CSV/XLSX 导入，复用现有 `import_job`、`import_job_row`、失败行修正、统一审核和审计能力。

## 范围边界

- 只导入来源资料元数据，不导入附件。
- 模板不得包含宗族 ID、支派 ID、来源资料 ID 等技术字段。
- 审核通过前只生成草稿来源资料；审核通过后进入正式来源资料库。
- 不新增独立任务表、错误行表或审核表。
