## 实现范围

- 390 px 小屏下三类桌面 Table 转为同数据源的响应式卡片记录视图；
- 卡片保留业务名称、范围、状态、关键摘要和权限动作；
- 小屏筛选保持单列，重置和查询按钮铺满并满足触控尺寸；
- 三类详情 Drawer 在文化页面小屏环境占满视口；
- Tabs 保持单行横向滚动；
- 页面头只展示一个主操作；
- 按钮、下拉入口和详情操作触控高度不小于 44 px；
- 编辑页维持单列，底部固定操作区适配安全区和换行；
- 桌面 Table、分页、URL 和权限状态不变。

## 非目标

- 不修改三类对象 API、数据状态和审核规则；
- 不新增独立移动端请求或第二套状态；
- 不引入新 UI 框架、PWA 或离线能力；
- 不重做桌面字段和详情内容。
